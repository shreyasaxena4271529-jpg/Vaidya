# Vaidya
Vaidya is an AI-powered healthcare assistant that replaces traditional, rigid chatbots with Active Medical Intelligence.

This project, titled Vaidya, is an advanced AI-powered knowledge base designed to transform how patients interact with healthcare information. Moving beyond traditional, rigid chatbots that often provide generic links, Vaidya utilizes Active Medical Intelligence to deliver precise, human-centered responses to complex healthcare FAQs.

Built on a Retrieval-Augmented Generation (RAG) architecture, the system parses user queries and retrieves verified data from trusted medical databases like SNOMED CT and PubMed. This ensures high accuracy (95%+) and eliminates "AI hallucinations," providing reliable guidance on symptoms, surgery preparation, and recovery.

Beyond simple Q&A, Vaidya acts as an Intelligent Agent capable of connecting to Electronic Health Records (EHR) and scheduling APIs to book appointments or verify insurance in real-time. By automating routine inquiries, the project reduces healthcare staff workload by up to 70%, offering patients 24/7 access to validated medical knowledge and personalized care instructions.
